Sobre Anna Bella

Modelo utilizada para fins de estudo sobre como utilizar tecnologias HTML e CSS em conjunto para desenvolver este site. Neste módulo de CSS aprendemos que o CSS pode ser utilizado de 3 maneiras: External style sheet, Internal style sheet e Inline, sendo o inline escrevendo o estilo dentro da própria TAG no Html, o internal escrito na TAG style dentro da TAG head e por último o External onde criamos um arquivo separado apenas para o CSS da página.

Temos então parte de seletores onde utilizamos a própria TAG para mencionar no CSS, ou Classes com o uso do . ou colocar ID utilizando então o # no CSS para se referenciar

Temos a TAG DIV que é genérica e utilizada para criar caixas(containers) dentro do HTML para facilitar na organização de conteúdo. A TAG span utilizamos para selecionar uma parte do conteúdo no qual queremos estilizar no CSS e com isso sera feito toda personalização no conteúdo que estiver dentro da TAG span.

Foi visto o conceito de bordas e seus estilos, fontes onde temos familia de fontes e tamanho, além de cores que podem estar em hexadecimal, RGB ou com o nome próprio da cor.

Em tamanhos vimos unidades como (px - pixel / % - relativo / em - relativo ao elemento pai / rem - outra unidade tambem relativa)
